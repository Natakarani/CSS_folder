defining color value

colors in css most often specifies is the following formulas

  1. a color keyword -like-"red","green","blue","transparent",etc
  2. A HEX value- like "#ff0000","00ff00","0000ff"etc
  3. an RGB value (red,blue,green)-like rgb(255,0,0)
